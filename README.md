# AquaManage-Android
AquaManage: An IoT-based technology to purify domestic wastewater for non potable uses

AquaManage is an innovative IoT-based technology designed to purify domestic wastewater for non-potable uses, contributing to sustainable water management. This repository contains the Android mobile application development for our CAPSTONE project, serving as the central control hub for AquaManage.

Features
1. Real-Time Monitoring: Track the status and performance of the wastewater purification process through live data feeds.
2. Remote Control: Manage and control the AquaManage system from anywhere using the mobile app.
3. Data Analytics: Analyze water quality and system efficiency with comprehensive data visualization tools.
4. Alerts and Notifications: Receive instant notifications for system alerts, maintenance reminders, and more.
